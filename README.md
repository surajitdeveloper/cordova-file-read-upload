# cordova-file-read-upload
This is includes
1. External file read by cordova
2. File Upload by angular http service
3. Angular integration
4. Angular Material Design
5. Angular Routing

You need 3 important plugin

1. Cordova file

to install     cordova plugin add cordova-plugin-file
weblink for reference:  https://cordova.apache.org/docs/en/latest/reference/cordova-plugin-file/

2. Cordova media capture

to install     cordova plugin add cordova-plugin-media-capture
weblink for reference:   https://cordova.apache.org/docs/en/latest/reference/cordova-plugin-media-capture/

3. Cordova permission

to install     cordova plugin add cordova-plugin-android-permissions
weblink for reference:    https://github.com/NeoLSN/cordova-plugin-android-permissions
